# Data inputs

This directory is intentionally empty.

The empirical analysis requires authorised local access to:

- TRACE corporate-bond transactions;
- TRACE Master File bond characteristics;
- CRSP issuer-equity data;
- public FRED market series.

TRACE, TRACE Master File and CRSP data were accessed through WRDS and cannot be
redistributed. Proprietary data, processed extracts and model-ready panels are
therefore not included in this repository.

Users with authorised access must provide their own local copies of the required
inputs. Expected filenames and required variables are documented in the source
code and project README.
