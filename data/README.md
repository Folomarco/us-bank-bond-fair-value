# Data inputs

This directory is intentionally empty.

The empirical analysis requires authorised local access to:

- TRACE corporate-bond transactions;
- TRACE Master File bond characteristics;
- CRSP issuer-equity data;
- public FRED market series.

TRACE, TRACE Master File and CRSP data were accessed through WRDS and cannot be
redistributed. Proprietary data, processed extracts and model-ready panels are
therefore not included in this repository.

Users with authorised access must provide their own local copies of the
restricted inputs using the following filenames:

- `us_banks_XX.zip` for the TRACE transaction extracts;
- `liquid_us_banks_bonds.zip` for the TRACE Master File extract;
- `us_banks_stocks.zip` for the CRSP issuer-equity extract.

Public FRED series are downloaded directly by `src/fred_data.py`, while the
market-holiday files are generated by
`src/create_market_holiday_files.py`.
